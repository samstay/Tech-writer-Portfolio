# TW Portfolio (GitHub Pages)

Готовый стартовый шаблон для портфолио технического писателя.  
**Как развернуть:** создайте репозиторий, загрузите файлы, включите GitHub Pages.

## Структура
```
.
├── _config.yml
├── index.md
├── about.md
├── resume.md
├── contact.md
├── assets/
│   ├── css/custom.css
│   └── img/sample-diagram.png
└── samples/
    ├── sample-1.md
    └── sample-2.md
```

## Локальный просмотр
Требуется Ruby/Jekyll, либо используйте предпросмотр GitHub. Для простоты можно пушить и смотреть прямо на Pages.
