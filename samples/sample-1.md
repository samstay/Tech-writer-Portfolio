## Контекст
Цель — помочь разработчику получить рабочий токен и сделать первый запрос за < 10 минут.

## Целевая аудитория
Backend‑разработчики, знакомые с HTTP и JSON.

## Предусловия
- Аккаунт в системе
- API ключ

## Шаги
1. Перейдите в **Настройки → API** и сгенерируйте ключ.
2. Сделайте тестовый запрос:

```bash
curl -X GET "https://api.example.com/v1/ping" -H "Authorization: Bearer <API_KEY>"
```

Ожидаемый ответ:
```json
{ "status": "ok" }
```
